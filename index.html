What is home automation?

It is a technological solution that enables automating the bulk of electronic, electrical and technology-based tasks within a home.
Early home automation began with labor-saving machines. 1975 first general purpose home automation network technology, X10.

X10 is a communication protocol for electronic devices. It primarily uses electric power transmission wiring for signalling and control, where the signals involve brief radio frequency bursts of digital data, and remains the most widely available.

A home automation system will control lighting, climate, entertainment systems, and appliances. It may also include home security such as access control and alarm systems. A home automation system typically connects controlled devices to a central hub or "gateway".

The home automation market estimated value $12.81 billion in the year 2020. (Expected to double by 2022)

Home automation suffers from platform fragmentation and lack of technical standards. The nature of home automation devices can also be a problem for security, data security and data privacy.
  • Electrical standards: voltage, frequency, plugs and sockets, battery shapes, etc... vs. Smart What?
  • Legal regulation (almost nothing)
  • Walled gardens:
    • Clouds: Apple Home kit vs. SmartThings, etc...
    • Standards: Zigbee vs. Z-Wave
    • Vendor lock in: no updates, forced purchases (new devices)
  • Failure, malfunction - inconvenient (standing before the door), challenging (who knows how to repair, rebooting samsung bluerays), just because internet outage
  • Bleeding edge versus leading edge:
    • Lighthouse cameras getting out of business
    • Nest purchase by Google => Nest Home no longer support IFTTT)
    • No budget for marketing, no budget for security
  • Security personal (listening, presence, pranks), vs. security IT (DDOS, network security, vendor lock in)
  • One compromised device could mean access to whole network

MicroPython
A microcontroller will run just one program repeatedly — not a full operating system.

MicroPython is a lean and efficient implementation of the Python 3 programming language that includes a small subset of the Python standard library and is optimised to run on microcontrollers and in constrained environments.

MicroPython is packed full of advanced features such as an interactive prompt, arbitrary precision integers, closures, list comprehension, generators, exception handling and more. Yet it is compact enough to fit and run within just 256k of code space and 16k of RAM.
